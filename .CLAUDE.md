## What It Is
Rust TUI + CLI (v0.1.5) for SSH key management.
Generate keys, manage targets, exchange public keys, run remote commands.
Config: `~/.config/cyberdeck/config.json`
Requires: `edition = "2024"`, `rust-version = "1.85"`

## Source Files (src/)
| File | Role |
|---|---|
| `main.rs` | CLI dispatch via Clap subcommands |
| `lib.rs` | Library re-exports |
| `models.rs` | `AppConfig`, `LocalKey`, `TargetProfile`, `AuthMethod` |
| `keys.rs` | `generate_ed25519_key()`, `scan_local_keys()`, `import_private_key()`, `expand_home_path()` |
| `ssh_ops.rs` | `exchange_public_key()`, `run_remote_command()`, `fetch_remote_authorized_keys()`, `test_connection()` |
| `storage.rs` | `load_config()`, `save_config()` ‚Äî JSON persistence |
| `tui.rs` | Full Ratatui TUI, crossterm raw mode, 4 tabs, event loop |

## CLI Subcommands (from main.rs)
```bash
cargo run                          # TUI (default, no subcommand)
cargo run -- tui                   # TUI (explicit)
cargo run -- list-keys             # list ~/.ssh keys
cargo run -- import-key --private-key <path> [--name <n>] [--passphrase <p>]
cargo run -- exchange --host <h> --port 22 --user <u> --public-key <k> [--key-file|--password]
cargo run -- run --host <h> --port 22 --user <u> --cmd <cmd> [--key-file|--password]
```

## TUI Structure (tui.rs)
4 tabs: `Keys`, `Targets`, `Exchange`, `Console`
- `MAX_LOG_LINES = 200`, `MAX_CONSOLE_LINES = 400`
- Crossterm raw mode with alternate screen
- State: `OnceLock<RwLock<...>>` for thread-safe config access
- Key bindings: `1-4` switch tabs, `q` quit, `F2` theme, `g` generate, `a` add target, `x` exchange, `e`/Enter run

## Hard Conventions
- SSH operations: use `ssh2` crate ‚Äî never `std::process::Command` with `ssh` binary.
- TUI state in `tui.rs` only. Business logic must not live in TUI code.
- Config: JSON at `~/.config/cyberdeck/config.json` ‚Äî do NOT migrate to TOML/YAML.
- `anyhow::Result` throughout ‚Äî no `.unwrap()` in non-test code.
- `--key-file` and `--password` args are mutually exclusive (`conflicts_with`).
