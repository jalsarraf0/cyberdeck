# .GEMINI.md - cyberdeck

This is a Rust application for SSH key management, featuring a Ratatui-based TUI and a supporting CLI for automation.

### Key Directories & Files
- `src/main.rs`: The main application entrypoint.
- `tests/regression.rs`: The core integration test suite.
- `scripts/regression.sh`: A comprehensive regression script that tests against a live, containerized SSH server.
- `scripts/install-local.sh`: A helper script to install the binary for the current user.
- `Cargo.toml`: Project manifest and dependencies.

### Core Commands
- `cargo build --release`: Compile the optimized binary.
- `cargo run`: Launch the interactive TUI.
- `cargo run -- --help`: View CLI commands and options.
- `bash scripts/install-local.sh`: Install the `cyberdeck` binary to `$HOME/.local/bin/`.

### Validation Commands
- `cargo test`: Run the unit tests.
- **`bash scripts/regression.sh`**: Run the full regression suite. This is the most important validation step. It builds the project, spins up a temporary SSH server in Docker, and runs a battery of integration tests against it.
- `cargo clippy -- -D warnings`: Run the clippy linter.
- `cargo fmt -- --check`: Check for formatting issues.
